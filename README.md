# 🧮 LaTeX Live Preview

一个基于浏览器的 LaTeX 公式实时预览与导出工具。写 LaTeX 不用再反复编译——敲下代码，右边马上看到效果。做 PPT、写笔记、搭博客的时候，直接把渲染好的公式导出成 PNG 或 SVG 贴进去就好。

这个项目最初只是一个简单的"输入公式 → 看效果"的页面，后来陆续加了很多东西：语法高亮、快速编辑工具栏、自定义宏命令、字体管理、图灵完备预处理器……算是一个比较完整的 LaTeX 在线编辑环境了。

## ✨ 能做些什么

- **实时预览**：输入 LaTeX 代码，右侧立刻渲染成 SVG 公式
- **语法高亮**：编辑器里会对 LaTeX 命令、括号、上下标做不同颜色的标记，写代码更舒服
- **快速编辑工具栏**：像 Word 里的公式编辑器那样，点按钮就能插入分数、根号、求和、积分、矩阵、希腊字母等
- **语法参考表**：忘了某个命令怎么写？点一下"语法参考"，200+ 条 LaTeX 命令按分类排列，点一下就能插入
- **放大查看**：点击预览区的公式可以弹出大图，支持鼠标滚轮缩放和拖拽平移，还能全屏
- **导出公式**：可以把公式导出成 PNG（支持透明/白/黑/自定义背景）或 SVG（矢量无限缩放）
- **换个字体**：内置几十种数学排版字体和系统字体，也能从 Google Fonts 加载、从 URL 导入、或直接拖拽字体文件上传
- **自定义命令**：像 LaTeX 的 `\newcommand` 一样定义自己的缩写宏，还能导出成 JSON 文件分享
- **图灵完备预处理器**：支持变量定义、条件判断、循环展开、算术运算，可以在"脚本模式"下生成动态公式（比如自动生成乘法表、斐波那契数列）
- **多语言模式**：除了 LaTeX，还预留了 AsciiMath 和 MathML 的输入模式切换
- **快捷键**：`Ctrl+S` 一键导出，`Ctrl+Shift+C` 复制 SVG 源码
- **偏好记忆**：你选的颜色、字体、背景、导出格式都会自动保存在浏览器里，下次打开还在

## 🧱 项目是怎么组织的

```
LeTexLivePreview/
├── index.html              # 入口页面，引用所有 CSS 和 JS
├── README.md               # 你现在看的这个文件
├── css/
│   └── style.css           # 所有样式写在一个文件里（按模块分区注释）
├── js/
│   ├── storage.js          # localStorage 偏好读写
│   ├── history.js          # 公式历史记录
│   ├── font-manager.js     # 字体管理（选择 + 导入 + UI 绑定）
│   ├── macros.js           # 自定义宏管理（定义 + 增删 + UI 绑定）
│   ├── preprocessor.js     # 图灵完备预处理器引擎 + UI 绑定
│   ├── latex-ref.js        # LaTeX 语法参考表 + UI 绑定
│   ├── syntax-highlight.js # 编辑器语法高亮
│   ├── quick-edit.js       # Word 式快速编辑工具栏
│   ├── preview-zoom.js     # 预览图点击放大（Canvas 位图 + 缩放 + 全屏）
│   ├── render.js           # MathJax 渲染管道（支持多输入模式）
│   ├── export.js           # PNG / SVG 导出 + 剪贴板复制
│   ├── ui.js               # 基础 UI 控件（背景/滑块/颜色/预设/折叠/快捷键）
│   └── main.js             # 全局状态 + 初始化 + MathJax 等待
└── assets/
    └── fonts/              # 本地字体文件存放目录（可选）
```

每个 JS 模块都自包含它的 UI 绑定逻辑——比如 `font-manager.js` 不仅管字体的数据加载，也绑定字体选择区域的按钮事件。这样改字体功能就只需要动一个文件。

## 🚀 怎么跑起来

不需要安装任何东西。直接用浏览器打开 `index.html` 就行，没有构建步骤，不依赖 Node.js。

唯一需要网络的地方是 MathJax 的 CDN（渲染引擎）和 Google Fonts（可选），不过都是国内可访问的镜像源。没网的话，系统内置字体也能用。

## 🔧 技术栈

- **MathJax 3** — LaTeX 渲染引擎（SVG 输出）
- **HTML Canvas** — PNG 导出 + 大图渲染
- **原生 JavaScript** — 全部逻辑，零框架依赖
- **CSS Variables + Grid** — 主题色 + 响应式布局
- **localStorage** — 用户偏好持久化
